#### https://personal-blog-mern-app.vercel.app/

>> This is the personal blog MernApp where an administrator can create, edit, and delete posts. Blog posts are written in Markdown for easy formatting and are rendered as HTML for public viewing. The admin interface is protected.
>>
>> ## Learning Outcomes

 The project needed to:

- Implement client‑side routing with React Router.

- Support full‑stack CRUD functionality.

- Add basic user authentication and protect routes.

- Handle and render Markdown content.

- Show a clean MERN application structure.

### << Grinding procedure>>
-> Set up the backend environment, initialize the Node.js server with Express, and establish a connection to the MongoDB database.
-> Design the data model and build the core REST API endpoints for managing blog posts without authentication.Build the public-facing pages of the blog: the main page listing all posts and the individual post page.
-> Integrate a Markdown parsing library to convert the post content into styled HTML for display.
-> Enhance the blog with SEO-friendly URLs, pagination for the post list, and dynamic page titles.Add the ability to categorize posts and allow users to filter the blog by a specific category.
->>Personal_blog-Mern-app-displayed.
